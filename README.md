# Mini4wdChrono
Arduino based 3-lane lap timer for Tamiya Mini4wd

## Develop on OSX

```
brew update
brew cask install arduino
// use the arduino IDE to upload StandardFirmataPlus on the board

brew install node
cd mini4wdchrono
npm install

// connect the device, then
npm start
```
